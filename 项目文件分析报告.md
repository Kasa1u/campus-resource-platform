# 校园课程资源共享平台 - 项目文件分析报告

## 📋 项目概述

本项目是一个**校园课程资源共享平台**，采用前后端分离架构：
- **后端**：Django + Django REST Framework + JWT 认证
- **前端**：Vue 3 + TypeScript + Vite + Element Plus + Pinia
- **数据库**：SQLite
