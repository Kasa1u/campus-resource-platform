## 💡 修改密码

所有账号都可以在**个人中心**修改密码：
1. 登录账号
2. 点击左侧菜单的"个人中心"
3. 点击"🔑 修改密码"按钮
4. 输入当前密码和新密码
5. 修改成功后需要重新登录

   - 已有 3 个示例帖子
   - 可以用不同账号发帖、评论，测试互动功能

## 🚀 快速开始

```bash
# 1. 启动后端
python3 manage.py runserver

# 2. 启动前端
npm run dev

# 3. 访问系统
# 前端：http://localhost:5173
# 后端：http://127.0.0.1:8000
# 管理后台：http://127.0.0.1:8000/admin
```

## 📧 账号信息导出

如需导出完整账号列表到 Excel 或其他格式，可以：
1. 登录 Django 管理后台
2. 进入"Users"应用
3. 选择"Export"功能
4. 导出 CSV 或 Excel 格式
